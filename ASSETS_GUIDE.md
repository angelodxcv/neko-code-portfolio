# Assets Guide - Where to Get Professional Pixel Art

## Japanese Street Background

**Recommended Sources (Free/Paid):**

1. **GuttyKreum - Japanese City (Free Version)**
   - https://guttykreum.itch.io/free-japanese-city-game-assets
   - Free tileset with Japanese buildings, streets, and elements
   - Download and use the background tiles

2. **GNDLF The Maker - 2D Background Pixel Art Japanese**
   - https://arexxuru.itch.io/pixel-art-japanese-background
   - 30 backgrounds in pixel art style
   - Perfect for RPG-style scenes

3. **Japanese PixelArt Cityscape Project**
   - https://sites.google.com/view/jp-pixel-city/home
   - Collaborative project with various Japanese street scenes

## Pixel Cat Sprite Sheet

**Recommended Sources:**

1. **ToffeeCraft - Cat Pixel Animations**
   - https://toffeecraft.itch.io/pixel-cat-animations
   - 32x32 pixel cat with multiple animations (walk, idle, sleep, etc.)

2. **Mattz Art - Cat 2D Pixel Art**
   - https://xzany.itch.io/cat-2d-pixel-art
   - 14 animations including walk, run, idle, attack

3. **DelineArte - Cat Animation Pack**
   - https://delinear-te.itch.io/cat-animation-pack
   - 126 sprites, 6 cat characters, 5 animations each

## How to Use

1. Download the assets from the links above
2. Place the background image in `assets/japanese-street-bg.png`
3. Place the cat sprite sheet in `assets/cat-sprite-sheet.png`
4. Update the CSS/JS with the correct sprite sheet dimensions
